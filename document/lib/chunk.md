# chunk

## 描述
将数组（array）拆分成多个 size 长度的区块，并将这些区块组成一个新数组。 如果 array 无法被分割成全部等长的区块，那么最后剩余的元素将组成一个区块。

## 参数
- array (Array) : 需要被拆分的数组。
- size (number) : 每个数组区块的长度

## 返回
-  返回一个包含拆分区块的新数组（注：相当于一个二维数组）

## 例子
```js
chunk([1, 2, 3, 4], 2)
// => [[1, 2], [3, 4]]

chunk([1, 2, 3, 4], 3)
// => [[1, 2, 3], [4]]

chunk([1, 2, 3, 4], 5)
// => [[1, 2, 3, 4]]
```
